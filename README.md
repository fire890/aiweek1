# retie - 은퇴자들의 이야기

**retie**는 은퇴한 분들이 자신의 소중한 경험과 지혜를 글로 공유하고, 다른 사람들과 소통할 수 있도록 만들어진 웹사이트입니다.

## 🌟 주요 기능

*   **손쉬운 글쓰기:** '새 글 쓰기' 버튼을 통해 누구나 쉽게 자신의 이야기를 작성하고 게시할 수 있습니다.
*   **아름다운 디자인:** 깔끔하고 현대적인 디자인으로 가독성을 높여 편안한 읽기 경험을 제공합니다.
*   **반응형 웹:** 데스크톱, 태블릿, 모바일 등 어떤 기기에서 접속해도 최적화된 화면을 볼 수 있습니다.

## 🛠️ 사용 기술

이 프로젝트는 별도의 프레임워크 없이 순수한 웹 표준 기술로만 제작되었습니다.

*   **HTML:** 웹 페이지의 구조를 담당합니다.
*   **CSS:** 현대적인 CSS 기능(Flexbox, Grid, CSS 변수 등)을 사용하여 디자인과 레이아웃을 구현합니다.
*   **JavaScript:** ES6+ 문법과 웹 컴포넌트(Web Components)를 사용하여 인터랙티브한 기능을 동적으로 구현합니다.

## 🚀 시작하기

1.  `index.html` 파일을 웹 브라우저에서 엽니다.
2.  화면에 나타난 '새 글 쓰기' 버튼을 클릭하여 새로운 글을 작성해보세요.
3.  게시된 글 목록을 확인합니다.
